Changelog

Progress you can track

ApiCourier is in active development. Major milestones and release notes will appear here as we ship.

Beta

April 2026 - Beta Milestone

Major expansion of protocol support and cross-platform readiness.

  • + macOS support in active beta (Universal binary: Intel + Apple Silicon)
  • + WebSocket client with full-duplex messaging and connection management
  • + gRPC client with server reflection and protobuf support
  • + Image preview in response viewer
  • + Keybinding service improvements and customization
  • ~ macOS-specific path handling and build fixes
  • ~ Trimmed assemblies for faster startup and smaller install
Beta

March 2026 - Feature Expansion

Core feature set buildout with scripting, import, and execution capabilities.

  • + GraphQL support with schema introspection
  • + Flow Runner with sequential execution, branching, and retries
  • + CLI tool (apicourier) with request runner, JUnit XML, and JSON reporters
  • + Code generation (C#, JavaScript, Python, cURL)
  • + Bruno (.bru) and cURL import capabilities
  • + Autocomplete for URLs and headers
  • + Middleware pipeline for request execution
  • + Local persistence layer
Foundation

Earlier 2026 - Foundation

Initial feature set establishing the Git-native, offline-first API client.

  • + Core HTTP client with all methods, headers, and auth (Bearer, Basic, API Key, OAuth2)
  • + Git integration (CLI + UI) with commit, branch context, status panel, and file watching
  • + OpenAPI 3.0/3.1 and Swagger 2.0 import
  • + Postman collection import (v2.0/v2.1)
  • + JavaScript scripting (ClearScript V8, Postman-compatible API)
  • + C# scripting (Roslyn-based with LINQ, records, pattern matching)
  • + Full environment management with variable substitution
  • + YAML-based Git-native collection storage
